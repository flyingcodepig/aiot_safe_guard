# somethingelse

This folder keeps non-core workspace materials out of the main project view.

Moved here:
- IDE and assistant-local settings: `.idea/`, `.claude/`
- Local runtime/cache materials: `venv/`, `backend/__pycache__/`
- Reference or vendored source snapshots: `llm_guard_src/`
- Older notes and reference docs: `readmes/`, `PROGRESS.md`, `README_FEATURES.md`, `USAGE.md`
- Historical phase artifacts: `response1_phase3.json`, `test_cases_phase3.json`, `test_cases_phase4_1.json`

Core project files remain in the repository root, `backend/`, `data/`, and `docs/`.
